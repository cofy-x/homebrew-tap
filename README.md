# cofy-x Homebrew Tap

Homebrew formulae for [cofy-x](https://github.com/cofy-x) projects.

## Install Elyro

Elyro supports Homebrew on macOS and Linux:

```bash
brew install cofy-x/tap/elyro
elyro version --json
```

Upgrade to the latest published version with:

```bash
brew update
brew upgrade cofy-x/tap/elyro
```

Project documentation, issues, and security reporting are maintained in the [Elyro repository](https://github.com/cofy-x/elyro).
